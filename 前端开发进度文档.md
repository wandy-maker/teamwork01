## 一、项目基本信息

| 项目 | 内容 |
| --- | --- |
| 项目名称 | 东软环保公众监督系统 |
| 前端负责人 | 待填写 |
| 前端范围 | 四端基础版前端开发 |
| 技术栈 | Vue3、Vite、Element Plus、Axios、Vue Router、ECharts |

## 四、接口预留说明

前端接口统一通过 Axios 请求层管理，便于后期统一修改后端地址、统一处理请求错误和登录状态。

接口约定如下：

| 约定项 | 内容 |
| --- | --- |
| 默认接口前缀 | `/api` |
| 查询类接口 | 使用 GET |
| 提交类接口 | 使用 POST |
| 请求数据格式 | JSON |
| 返回数据格式 | `{ code, message, data }` |
| 成功状态码 | `code === 200` |
| 失败处理 | 使用 Element Plus 消息提示展示错误信息 |

建议后续前端新增以下接口文件：

| 文件 | 负责模块 |
| --- | --- |
| `src/api/request.js` | Axios 基础配置 |
| `src/api/supervisor.js` | 公众监督员注册、登录、信息查询 |
| `src/api/aqiFeedback.js` | 空气质量反馈提交、查询、指派 |
| `src/api/gridMember.js` | 网格员登录、网格员查询 |
| `src/api/statistics.js` | AQI 检测确认和统计数据 |
| `src/api/admins.js` | 管理员登录 |
