## 一、项目概述

东软环保公众监督系统（Neusoft Environmental Public Supervision System）是一个面向网格员的移动端Web应用，用于接收公众空气质量反馈、实地检测并提交AQI实测数据。

```
└────────────────────────┬────────────────────────────┘
                         │ HTTP POST (JSON)
                         │
┌────────────────────────┴────────────────────────────┐
│                  后端 (Spring Boot)                   │
│         http://localhost:8080/nepm                  │
```

## 四、后端启动

### 4.1 配置数据库连接

编辑 `02.后端/src/main/resources/application.yml`：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/nep?useSSL=false&serverTimezone=Asia/Shanghai&characterEncoding=utf-8
    username: root
    password: 1074382447    # <-- 修改为你的 MySQL root 密码
```

### 4.2 启动后端

```powershell
# 进入后端目录
cd 02.后端

# 首次运行需要下载依赖（约 3-5 分钟）
mvn spring-boot:run
```

启动成功标志：

```

  访问地址: http://localhost:8080/nepm
======================================
Tomcat started on port(s): 8080
```

```powershell

curl -X POST http://localhost:8080/nepm/gridMember/getGridMemberByCodeByPass `
  -H "Content-Type: application/json" `
  -d '{"gmCode":"caocao","password":"123"}'
```

|------|-----|------|

### 6.2 反馈任务列表

登录后进入反馈任务列表页面，显示已指派给当前网格员的反馈信息：

- 彩色方块 = 公众预估的AQI等级（绿→黄→橙→红→紫）
- 显示省份、城市、地址、反馈日期
- 点击"去检测"进入实测页面

### 6.3 实测数据提交

在反馈详情页：

1. 查看反馈信息（监督员、区域、预估等级、描述）
2. 输入三项实测浓度值：
   - **SO₂浓度**（二氧化硫，μg/m³）
   - **CO浓度**（一氧化碳，μg/m³）
   - **PM2.5浓度**（悬浮颗粒物，μg/m³）
3. 系统自动根据AQI标准计算各项等级和综合等级
4. 点击"提交实测数据"保存

### 6.4 业务状态流转

```
公众监督员提交反馈 → 状态: 未指派(0)
                        ↓ 管理员指派
                  状态: 已指派(1)
                        ↓ 网格员检测提交
                  状态: 已确认(2)
```

## 七、常见问题

### Q1: 数据库连接失败

**错误**: `Access denied for user 'root'@'localhost'`

**解决**:
1. 确认 MySQL 服务已启动: `Get-Service MySQL84`
2. 核对 `application.yml` 中的密码
3. 尝试用 MySQL Workbench 测试连接

### Q2: 端口被占用

**错误**: `Port 8080 is already in use`

**解决**:
```powershell
# 查找占用 8080 端口的进程
netstat -ano | findstr :8080
# 杀掉进程（替换 PID）
taskkill /PID <PID> /F
```

### Q3: 前端跨域错误

**错误**: `Access-Control-Allow-Origin` 相关错误

**解决**: 确认后端已启动且 `CorsConfig.java` 配置正确

### Q4: 登录后无反馈数据

**原因**: 该网格员没有被指派反馈任务

**解决**:
```sql
-- 查看网格员已指派的反馈
SELECT af_id, tel_id, address, state 
FROM aqi_feedback 
WHERE gm_id = <你的gm_id> AND state = 1;

-- 手动指派反馈给网格员
UPDATE aqi_feedback 
SET gm_id = <你的gm_id>, state = 1, assign_date = CURDATE(), assign_time = CURTIME()
WHERE af_id = <反馈ID>;
```

### Q5: npm install 失败

**解决**:
```powershell
# 清除缓存重试
npm cache clean --force
npm install

# 或使用淘宝镜像
npm install --registry=https://registry.npmmirror.com
```

### Q6: Maven 依赖下载慢

**解决**:
在 `~/.m2/settings.xml` 中添加阿里云镜像：
```xml
<mirrors>
  <mirror>
    <id>aliyun</id>
    <mirrorOf>central</mirrorOf>
    <name>Aliyun Maven</name>
    <url>https://maven.aliyun.com/repository/public</url>
  </mirror>
</mirrors>
```
