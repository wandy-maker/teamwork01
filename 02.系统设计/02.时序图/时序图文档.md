# 东软环保公众监督系统 — 时序图文档

## 1. 网格员登录流程

```
公众监督员(网格员)    Vue前端         SpringBoot后端      MySQL数据库
     |                 |                  |                  |
     |--输入登录编码---->|                  |                  |
     |  和密码          |                  |                  |
     |                 |--POST /gridMember|                  |
     |                 |  /getGridMember--->|                  |
     |                 |  ByCodeByPass    |                  |
     |                 |    {gmCode,      |                  |
     |                 |     password}    |                  |
     |                 |                  |--SELECT * FROM   |
     |                 |                  |  grid_member     |
     |                 |                  |  WHERE gm_code=? |
     |                 |                  |  AND password=?  |
     |                 |                  |  (含JOIN省/市表)  |
     |                 |                  |<--GridMember对象--|
     |                 |<--JSON(GridMember)|                 |
     |                 |                  |                  |
     |                 |--存入session---->|                  |
     |                 |  Storage         |                  |
     |                 |--路由跳转到------>|                  |
     |                 |  /feedbackList   |                  |
     |<--显示反馈列表页--|                  |                  |
```

## 2. 查询反馈任务列表流程

```
网格员              Vue前端             SpringBoot后端        MySQL数据库
 |                    |                       |                    |
 |                    |--页面加载自动调用------>|                    |
 |                    |  init()               |                    |
 |                    |--POST /aqiFeedback    |                    |
 |                    |  /listAqiFeedback----->|                    |
 |                    |  ByGmId               |                    |
 |                    |  {gmId, state:1}      |                    |
 |                    |                       |--SELECT * FROM     |
 |                    |                       |  aqi_feedback af   |
 |                    |                       |  JOIN supervisor   |
 |                    |                       |  JOIN grid_province|
 |                    |                       |  JOIN grid_city    |
 |                    |                       |  JOIN aqi          |
 |                    |                       |  WHERE af.gm_id=?  |
 |                    |                       |  AND af.state=1    |
 |                    |                       |  ORDER BY af_date  |
 |                    |                       |  DESC, af_time DESC|
 |                    |                       |<--List<AqiFeedback>|
 |                    |<--JSON数组------------|                    |
 |                    |                       |                    |
 |                    |--渲染列表(颜色标签+---->|                    |
 |                    |  省/市/日期/地址+      |                    |
 |                    |  "去检测"按钮)         |                    |
 |<--显示反馈列表------|                       |                    |
```

## 3. 实测数据提交流程（核心业务）

```
网格员              Vue前端               SpringBoot后端        MySQL数据库
 |                    |                        |                    |
 |--点击"去检测"------>|                        |                    |
 |                    |--POST /aqiFeedback     |                    |
 |                    |  /getAqiFeedbackById-->|                    |
 |                    |  {afId}                |                    |
 |                    |                        |--SELECT af.*,     |
 |                    |                        |  s.real_name,      |
 |                    |                        |  p.province_name,  |
 |                    |                        |  c.city_name,      |
 |                    |                        |  a.chinese_explain |
 |                    |                        |  FROM aqi_feedback |
 |                    |                        |  af                |
 |                    |                        |  JOIN supervisor s |
 |                    |                        |  JOIN grid_province|
 |                    |                        |  p                  |
 |                    |                        |  JOIN grid_city c  |
 |                    |                        |  JOIN aqi a        |
 |                    |                        |  WHERE af.af_id=?  |
 |                    |                        |<--AqiFeedback对象--|
 |                    |<--JSON(完整反馈详情)----|                    |
 |                    |                        |                    |
 |                    |--并行请求: POST /aqi    |                    |
 |                    |  /listAqiAll----------->|                    |
 |                    |                        |--SELECT * FROM aqi|
 |                    |                        |  ORDER BY aqi_id  |
 |                    |                        |<--List<Aqi>-------|
 |                    |<--JSON(AQI等级数组)-----|                    |
 |                    |                        |                    |
 |<--显示反馈详情+实测--|                        |                    |
 |  输入表单           |                        |                    |
 |                    |                        |                    |
 |--输入SO2/CO/PM2.5->|                        |                    |
 |  浓度值             |                        |                    |
 |                    |--前端computed计算------>|                    |
 |                    |  (根据aqiArr阈值       |                    |
 |                    |   匹配各指标等级+颜色)   |                    |
 |                    |                        |                    |
 |--点击"提交实测数据"->|                        |                    |
 |                    |--表单验证(非空+数字)---->|                    |
 |                    |--confirm确认框--------->|                    |
 |                    |                        |                    |
 |                    |--POST /statistics      |                    |
 |                    |  /saveStatistics------->|                    |
 |                    |  {全部实测参数}          |                    |
 |                    |                        |--INSERT INTO       |
 |                    |                        |  statistics(...)   |
 |                    |                        |<--影响行数=1-------|
 |                    |<--1(成功)--------------|                    |
 |                    |                        |                    |
 |                    |--POST /aqiFeedback     |                    |
 |                    |  /updateAqiFeedback---->|                    |
 |                    |  State                 |                    |
 |                    |  {afId, state:2}       |                    |
 |                    |                        |--UPDATE            |
 |                    |                        |  aqi_feedback      |
 |                    |                        |  SET state=2       |
 |                    |                        |  WHERE af_id=?     |
 |                    |                        |<--影响行数=1-------|
 |                    |<--1(成功)--------------|                    |
 |                    |                        |                    |
 |                    |--router.push----------->|                    |
 |                    |  ('/feedbackList')     |                    |
 |<--返回反馈列表页----|                        |                    |
```

## 4. 数据流总结

```
┌─────────────────────────────────────────────────────────────┐
│                        用户角色                               │
├──────────────┬──────────────────┬────────────────────────────┤
│  公众监督员   │    网格员         │     系统管理员              │
│ (supervisor) │  (grid_member)   │    (admins)                │
├──────────────┼──────────────────┼────────────────────────────┤
│ 提交反馈信息  │ 查看已指派反馈     │ 管理网格员/指派任务         │
│ (aqi_feedback │ 输入实测数据      │ 查看统计数据               │
│  state=0)    │ (statistics)     │                           │
│              │ 确认完成(state=2) │                           │
└──────────────┴──────────────────┴────────────────────────────┘

反馈状态流转: 未指派(0) ──管理员指派──> 已指派(1) ──网格员检测──> 已确认(2)
```
